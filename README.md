# 📦 Simple ERP System

A simple ERP system developed with a focus on best programming practices and the creation of a functional, well-structured, and easy-to-use application.

---

## 📖 About the Project

This project was created with the goal of studying and applying modern software development practices through the construction of a real-world ERP system.

It is being developed using **PHP with the Laravel framework**, aiming to create a real production environment while reinforcing important development concepts.

## ⚙️ Installation

> **Requires:** Docker

### 1. Clone the repository

```bash
git clone https://github.com/LeandroMF20/Simple-OS.git
cd Simple-OS
```

### 2. Run the installer

```bash
./install.sh
```

The installer will automatically:
- Copy the `.env` files from the examples
- Build and start the Docker containers (PHP + MySQL)
- Wait for the database to be ready
- Install Composer and npm dependencies
- Generate the application key
- Run the database migrations

Once finished, the application will be available at **http://localhost:8080** and MySQL database will be available at **localhost:3306**.
