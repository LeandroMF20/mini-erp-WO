# 📦 Simple ERP System

A simple ERP system developed with a focus on best programming practices and the creation of a functional, well-structured, and easy-to-use application.

---

## 📖 About the Project

This project was created with the goal of studying and applying modern software development practices through the construction of a real-world ERP system.

It is being developed using **PHP with the Laravel framework**, aiming to create a real production environment while reinforcing important development concepts.

---

## 🚧 Project Status

⚠️ This project is currently in an **early stage of development**.

At the moment, the focus is on building a solid foundation before moving forward to more advanced features.

---
